# bear-down-for-midterms

*Bear Down for Midterms*

Bear Down for Midterms is a Pomodoro Timer app designed to keep students motivated. Users complete tasks, earn bricks, and build a bear village as they study.

*Features*

Pomodoro Timer for focus and break cycles
Task List to manage study goals
Village Building: Earn bricks for each task and build houses for bears (10 bricks = 1 house)

*Usage*

Home: Navigate to all features.
Pomodoro Timer: Set tasks and work in cycles.
Village: Track progress by building houses with earned bricks.

*Tech Stack*

HTML, CSS, JavaScript

*Future Plans*

More village elements (gardens, parks)
User login for data saving
Enhanced gamification
